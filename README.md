# MK_CaptureAndReplay-MouseKeyboard-
基于QT5.14.2_MSVC2017_32bit的环境，采用多线程编程开发了一款能够捕获用户在当前电脑环境下的鼠标键盘事件，并采用JSON格式记录，同时也可以选择指定目录下加载先前用户记录的事件记录数据，配置事件回放参数，可以完成对先前操作的复现实现无聊琐碎的工作（ps: Ciallo～ (∠・ω&lt; )⌒★别太指望能够完全复现，很有可能由于当前电脑环境不同于之前的而导致回放失败，比如之前想要打开b站，但是由于现有网速较慢，导致网页没能打开，后续的操作没能衔接上而失败，问题不大但是点错到其他地方就不好了，慎用！）。

软件操作说明：

打开：MouseKeyboardCaptureExe压缩包解压后，直接点击可执行程序.exe文件即可，然后会弹出两个窗口，控制台(黑)和软件界面(白)，控制台仅用于调试和检查问题，如果打开没问题就可以开始录制了。

操作录制：在界面中点击左上角的btnStart，首先会弹出一个文件选择路径，选择一个你想要保存录制信息的目录，并未你的操作录制文件命名即可(仅需要给名称如(XXX事件或XXX操作等)不需要后缀)。然后就可以开始录制在用户当前电脑环境下的各种鼠标键盘的操作，界面中间会不断地刷新用户所作出的操作。最后点击左上角的btnStart右边的btnStop按钮即可停止录制，再操作回放过程中并不会将此（停止录制）操作也进行复现，所以不用担心。

操作复现：在软件界面往下看，会有回放的控制面板，可以选择要回放的文件，以及清空当前选择的文件clear、需要回放的事件和回放速度，一般不建议修改回放速度。选择好之前录制好的回放文件，点击控制面板中的启动回放按钮即可放开鼠标和键盘，让电脑自行操作吧。

回放控制快捷键：在回放的过程中如何控制呢，那就需要我们使用之前注册好的热键，默认热键为(停止回放：Ctrl+Alt+S、暂停回放：Ctrl+Alt+P、继续回放：Ctrl+Alt+R、倍速切换：Ctrl+Alt+A)。
